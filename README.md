### everything in the commands section is optional
---
To run this code keep in mind you need a folder named "bigData" in the same dir

```plaintext
main.lua
--> bigData
--> --> [*.txt]
```

inside of bigData add text files add as much text/files as you can, the more data it has the better answers it gives, though **it doesn't think**

***Basically*** it works like your phone autofill, searches for what you typed, checks in the files and sees which words usually goes next, not a single thought in the process.
<hr><br>
The examples given inside `bigData` are really chaotic, I suggest you start from scratch for better results.
You can try pasting some files you have that *you* wrote then see if it matches how you speak.
